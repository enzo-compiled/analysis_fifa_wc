## Análisis del dataset FIFA WC (1930-2018).
---
La idea del proyecto es explorar el dataset FIFA WC. Para ello, se hizo una carga de los datasets, una limpieza, un eda y finalmente se entrenó un modelo logístico multinomial 70/30,
con el fin de predecir el ganador de un partido a partir de una variable categórica `outcome (H/A/D)`.
